課題2
  コマンドプロンプトで make と入力して、実行可能ファイルが
全てコンパイルできることを確認してください。
  もう一度 make でコンパイルし、更新されたファイルがなければ
コンパイルしないことを確認してください。
  touch コマンドでどれか1つのソースファイルの更新時刻を現在時刻に
変更してから make してください。
  いくつかのソースファイルに対して同様に試してください。
  make clean で実行可能ファイルとオブジェクトファイルが削除される
ことを確認してください。

課題3
  ターミナルを2個(AとBと呼ぶ)起動してください。

  ターミナルA にて、
    $ ./TCPEchoServer ポート番号
を実行してください。
ここで、$ はコマンドプロンプトを示すので $ より右を入力してください。
また、ポート番号は、学籍番号のハイフンなし の値としましょう。
    例）23-401 の場合
    $ ./TCPEchoServer 23401

  ターミナルB にて、
    $ ./TCPEchoClient 127.0.0.1 "test" ポート番号
を実行してください。
サーバを起動しているターミナルA に
    Handling client 127.0.0.1
と表示され、クライアントを起動したターミナルB に
    Received: test
と表示されることを確認してください。
  何回か、クライアントから異なる文字列を送信してみてください。
